## 4-week PR timeline (1 peg per week)

### Week 1 — Foundation
- Presskit page `/press` complete (factsheet + links + 3 raw clips)
- Create 6–10 screenshots set
- Create “how to record a clip” note for playtesters (optional)
- Publish 3–5 short clips (7–12s) across socials

### Week 2 — First peg: demo / vertical slice
- Announce: “Demo live” / “Playtest weekend”
- Outreach block (60–90 min) to Brabant outlets using forms/tip pages
- Post 1 trailer (30–60s) + 3 raw clips

### Week 3 — Proof + community angle
- Post a small truthful update: #playtesters, #Discord, #issues closed (if available)
- Run 1 community event (“dev plays with viewers” / “Brabant vs Randstad” showmatch)
- Second outreach wave: “community builds it live”

### Week 4 — Second peg: major update
- Ship a noticeable update (new map/unit/polish/perf)
- Refresh presskit with new clip/screenshot
- Re-contact non-responders with “what’s new”

### Rules
- Keep every message “local + visual + now”: Brabant hook, clip, link, CTA.
