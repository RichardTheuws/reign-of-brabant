# Belgen — Vrouwelijke Vlaams-Belgische Spreker

**138 lines totaal** (8 units × 15 + 18 generic fallback). Vlaamse zuiderbuur-toon — warm, melodieus, "ge/gij", "amai", "allez", "voila".

## Karakterregie (lees eerst)

| Unit | Karakter |
|---|---|
| **frietkraamhouder** (worker) | Vlaamse frietvrouw; warm, hartelijk: "allez schat, een frietje?". Markt-energie, schort, gulle stem. |
| **bierbouwer** (infantry) | Abdijbrouwerin; sereen-meditatief, bier als gebed. Zacht-articulerend, eerbiedig — denk Trappisten-non. |
| **chocolatier** (ranged) | Brusselse chocolatière; verfijnd, ietwat parijs-Brussels accent. Articulatie als pralinedoosje openen. |
| **frituurridder** (heavy) | Vrouwelijke vetridder; trots op het frietkraam-erfgoed. Stevige bek, schort als wapenrusting. |
| **manneken-pis-kanon** (siege) | Magritte-Brussel absurdisme; "ceci n'est pas un kanon"; theatraal, surrealistisch, droog-grappig. |
| **wafelzuster** (support) | Religieuze warmte; abdij-zuster-energie. Wafelijzer = altaar. Zacht, dragend, troostend. |
| **dubbele-spion** (special) | Fluisterend-glad, mata-hari-vibe. Nooit duidelijk welke kant ze kiest. Frans-Vlaams door elkaar. |
| **de-frietkoning(in)** (hero) | **Frietkoningin**; "In naam van de frieten…". Plechtig, regaal, maar met saus aan haar kroon. |

**Toon-referentie**: bestaande Petra (jong/casual) en Sharon (jong/articulate). Deze scripts liggen ouder/karaktervoller — pak een rolletje per unit, niet één neutrale stem.

## Hoe op te nemen

- Open **Voice Memos** (iPhone) of **QuickTime Player** (Mac, "New Audio Recording")
- Eén bestand voor alles — geen knip-tussen-units
- **Per regel**: zeg eerst het **nummer hardop** ("één", "twee"… "honderdachtendertig"), dan **2-3 seconden stilte**, dan de **tekst**
- **Hertake?** zeg de tekst opnieuw — splitter pakt de laatste versie per slot
- **Volgend nummer**: laat eerst minstens **1 seconde stilte** vallen na de laatste take

**Tekst is suggestief — schrijf ter plekke je eigen versie.** Eis: behoud **bedoeling/intentie** per slot (een select klinkt anders dan een attack). Volgorde MOET kloppen, anders mappen files verkeerd.

Levering: `recording-belgen-female.wav`/`.m4a`/`.mp3` naar `~/Documents/games/reign-of-brabant/_recording/`.

## Levering & splitter

1. Sla op als `_recording/recording-belgen-female.wav` (of `.m4a`/`.mp3`)
2. `bash scripts/split-voice-recording.sh _recording/recording-belgen-female.wav RECORDING-SCRIPT-BELGEN-FEMALE.md`
3. Splitter detecteert stilte tussen segmenten, neemt **laatste take per slot**, schrijft naar `public/assets/audio/voices/belgen/...` (sub-folder per unit + generic pool — kan via flag `--sub female-2` als extra female voice naast Sharon)
4. Daarna `bash scripts/normalize-voices.sh --all` voor uniforme loudness
5. Audit-page: open `voices-audit-belgen.html` om alle 138 te beluisteren
