## Research (working documents)

This folder contains **iterating research and working documents** for `reign-of-brabant.nl` (crowdfunding, PR, site IA, and roadmap).

If a document becomes “canonical” (spec/PRD), move or summarize it into `docs/`.

### Structure
- `crowdfunding/`: payment provider notes, copy drafts, privacy notes, operational checklists
- `pr/`: media lists, pitch templates, outreach sequences, presskit checklist
- `roadmap/`: milestone ladder, improvement backlog, risks/acceptance criteria
- `site-ia/`: information architecture, URL map, deploy notes, test checklists
